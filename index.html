    FITS Image Viewer

    Project maintained by badders Hosted on GitHub Pages — Theme by mattgraham

    pyfitsview

    A basic image viewer for FITS images based on aplpy and Qt4. Optionally supports directly downloading images from the SBIG AllSky Camera using pyallsky

    Requirements

    • Qt library v4
    • Python 2.7

    Python package requirements

    • pyallsky (Optional)
    • PyQt4
    • NumPy
    • matplotlib
    • astropy
    • APLpy
